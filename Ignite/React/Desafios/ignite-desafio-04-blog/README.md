## Desafio Ignite React - 04
Criar do zero uma aplicação de um blog com Prismic CMS a partir de um layout do Figma

&nbsp;
## ⚙️ Test
Para compilar a aplicação utilize a sequência: 
```
yarn
yarn dev
```

Para testar a aplicação utilize:
```
yarn test
```
