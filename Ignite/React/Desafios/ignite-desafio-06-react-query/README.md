## Desafio Ignite React - 06
Realizar o upload e listagem de imagens utilizando React Query, React Hook Form e Chakra UI.

&nbsp;
## ⚙️ Test
Para compilar a aplicação utilize a sequência: 
```
yarn
yarn dev
```

Para testar a aplicação utilize:
```
yarn test
```
